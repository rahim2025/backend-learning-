# Backend Learning with JavaScript

This repository contains my learning notes, practice examples, and interview preparation materials for software backend development using JavaScript.

The goal of this repo is to build a strong foundation in backend concepts, JavaScript behavior, validation, APIs, and real-world problem solving.

## What This Repo Covers

- JavaScript fundamentals for backend development
- Type coercion and validation edge cases
- Backend interview preparation
- Practical bugs that happen in server-side JavaScript
- Clean notes that can be reviewed before interviews

Type coercion is important in backend development because request data often comes from query strings, request bodies, forms, and external APIs. These values may arrive as strings, missing values, empty values, or unexpected types.
