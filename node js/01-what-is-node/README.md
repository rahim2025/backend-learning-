# What is Node.js?

Node.js is a JavaScript runtime environment.

It allows JavaScript to run outside the browser.

```txt
Node.js = JavaScript runtime for running JavaScript outside the browser
```

## What Runtime Means

A runtime is an environment where code can execute.

Node.js gives JavaScript access to backend features such as:

- File system
- Network
- HTTP server
- Timers
- Streams
- Operating system utilities

## Example

```js
console.log("Hello from Node.js");
```

Run:

```bash
node app.js
```

## Interview Answer

Node.js is a runtime environment that lets JavaScript run outside the browser. It is commonly used to build backend servers, APIs, command-line tools, and real-time applications.
